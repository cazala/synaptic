Synapse.js
Synapse.js is an architecture-free neural network library for node.js and the browser.
creature.js
var synapse = require('synapse'); this.network = new synapse.Architect.Perceptron(40, 50, 3);
world.js
creatures.forEach(function(creature) { // move var input = []; for (var i in creatures) { input.push(creatures[i].location.x); input.push(creatures[i].location.y); input.push(creatures[i].velocity.x); input.push(creatures[i].velocity.y); } var output = creature.network.activate(input); creature.moveTo(output); // learn var learningRate = .9; var target = [ targetX(creature), targetY(creature), targetAngle(creature)]; creature.network.propagate(learningRate, target); });
In this demo the creatures learn how to move all together as a group.
Neuron

Neurons are the basic unit of the neural network. They can be connected to another neuron or gate connections between other neurons. This allows you to create complex and flexible architectures.

+ Read documentation

Network

Networks can be imported/exported to JSON, converted to workers or standalone functions. They can be connected to other networks or gate connections.

+ Read documentation

Trainer

Trainers can take any given network regardless of its architecture and use any training set. It includes built-in tasks to test networks, like learning an XOR, completing a Discrete Sequence Recall task or an Embeded Reber Grammar test.

+ Read documentation

Architect

The Architect includes built-in useful architectures like multilayer perceptrons, multilayer long-short term memory networks (LSTM) and liquid state machines.

+ Read documentation
